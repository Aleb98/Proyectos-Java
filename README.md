# CerrajeroSeguro
- Esta aplicacion da soporte a servicios de la cerrajeria de un amigo. Se esta terminando, por eso sigue en desarrollo. Da soporte a la gestion de venta del negocio, 
  gestion de usuario e inventario.
  
## Caracteristicas Principales
- Registro y gestion de ventas de cerrajeria.
- Persistencia de datos en **MySQL**.
- Arquitectura en capas con principios **SOLID**.
- Interfaz simple y orientada a la usabilidad.
